Multiline blockquotes with a cite reference:

The HTML < blockquotes > Element (or HTML Block Quotation Element ) indicates that the enclosed text is extended quotation. Usually this is rendered visually by indentation (see Notes for how for to change it). A URL for the source of the quotation may be given using the cite attribute, while a text representation of the source can be using the < cite > element.
